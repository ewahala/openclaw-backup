# HEARTBEAT.md - NEO持续工作配置

# Heartbeat每30分钟触发一次，执行以下任务：

## 持续任务清单

### 1. 调研学习 (每次必做)
- 搜索AI硬件/玩具市场新产品动态（web_search API到限额时用无头浏览器）
- 访问行业报告网站获取市场数据
- 关注Kickstarter/Indiegogo众筹新品（被Cloudflare阻挡时可跳过）
- 收集行业趋势和技术方向

### 2. 知识积累
- 更新AI产品调研报告 (memory/projects/ai-product-research.md)
- 记录新发现的产品、趋势、供应商
- 更新MEMORY.md中的调研发现

### 3. 自我学习
- 每天完整阅读一次OpenClaw文档（检查更新）
- 学习新的调研方法和工具
- 探索新的技能(skills)来提升调研能力

### 4. 产品方向深耕
- 每周深入分析3-5个竞品
- 评估技术方案和BOM成本
- 探索销售渠道和众筹策略

## 执行原则

1. 每次heartbeat至少完成一项调研任务
2. 重要发现立即记录到memory
3. 持续保持学习状态，24小时不间断
4. 凌晨1点(北京时间)除了日常任务，还需进行深度复盘

## 输出要求

- 每次heartbeat后更新 memory/YYYY-MM-DD.md
- 重要发现更新 MEMORY.md
- 每天产出至少一条有价值的调研发现

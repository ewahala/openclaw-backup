- **点赞**: 12小时内≥1000，累计≥5000

### 2026标题趋势

- 口语化表达
- 制造冲突感/悬念

## 平台算法深度（2026-03-05 11:45）

### 星云5.0核心
- 从"停留时长"→"价值内容匹配"
- 五大引擎：用户画像、内容理解、匹配排序、流量调控、商业平衡

### 流量分发逻辑
1. 先审核收录 → 2. 打标签匹配 → 3. 阶梯式流量池 → 4. CES评分 → 5. 搜推互通

### 三阶流量测试
- 初始曝光: 100-500展现，CTR≥3%
- 互动率筛选: ≥5%
- 全站分发: 进入更大流量池

### 内容权重公式
推荐得分 = 基础质量分×0.3 + 用户互动分×0.4 + 商业价值分×0.2 + 社交关系分×0.1 - 违规扣分

### 关键阈值
- CTR: 图文≥2.5%，视频≥4%
- 视频特权: 初始曝光比图文高30%，完播率>50%获二次推荐

### 2026新趋势

#### 1. 站内闭环崛起
- **种草→下单→站内支付** 全链路在小红书完成
- 适合高颜值、上新频繁、客单价低的产品

#### 2. 全域经营（必须）
- 笔记种草 + 直播转化 + 私域复购 = 闭环
- 单一渠道无法持续赚钱

#### 3. 私域新选择
- 微信不再是唯一
- 小红书站内私域（群聊、橱窗）也能承接

### 私域SOP（经典版）
1. **养号**: 微信小号提前3天养
2. **钩子**: 自动回复设成送资料包
3. **标签**: 企业微信SCRM打好标签
4. **转化**: 48小时内做低客单转化
5. **及格线**: 日加100+精准用户

### 细分赛道逻辑
- 竞争小、用户精准
- 做深做透才能长久生存
- 浅尝十个大赛道不如吃透一个细分

### 变现路径
- **接广**: 蒲公英平台，品牌合作
- **带货**: 站内闭环或站外淘宝京东
